El protocolo HTTP y el estilo arquitectónico REST

Introducción

HTTP (Hypertext Transfer Protocol) es el protocolo de la web. Es un protocolo muy rico, muchas veces muy poco aprovechado. Sin lugar a dudas la web es un caso de éxito en tecnología, sin ella no podemos concebir el mundo tecnológico tal y como es hoy en día.

REST (Representational State Transfer) es un estilo arquitectónico para el desarrollo de aplicaciones Web. Su principal característica es que hace un uso completo del protocolo HTTP.

En este capítulo revisaremos el protocolo HTTP, veremos la filosofía del estilo arquitectónico REST, los niveles de una aplicación web para que sea RESTful, y cómo crear aplicaciones REST con jersey
